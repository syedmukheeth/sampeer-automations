# Brand assets

The app loads the Sampeer Studio logo from this folder.

- **`SampeerStudio-Logo.png`** - the real Sampeer Studio logo. It is used in the sidebar, login page, and browser tab.

Recommended: square, transparent or black background, 256x256 or larger.

All UI references `/SampeerStudio-Logo.png`. If the image is removed, the app falls back to a styled letter mark instead of rendering a broken image.
